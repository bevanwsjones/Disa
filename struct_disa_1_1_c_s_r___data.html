Detailed Description

template<typename _value_type, typename _index_type = std::size_t>
struct Disa::CSR_Data< _value_type, _index_type >

A structure to represent Compressed Sparse Row (CSR) data format.

This structure is used to store sparse matrices in CSR format, it contains only the data and no operations. The data is stored in three standard vectors: row_offset, column_index, and value. The row_offset vector stores the index of the first non-zero element in each row, the column_index vector stores the column index of each non-zero element, and the value vector stores the value of each non-zero element. The columns member variable stores the number of columns in the matrix.

The class is not intended for use outside of the Matrix_Sparse class which wraps it.

Template Parameters
_value_typeThe type of the values stored in the matrix.
_index_typeThe type of the indices used for rows and columns, defaults to std::size_t.
