\doxysubsection{Detailed Description}
Contains the criteria values against which convergence status can be assessed. 

This struct holds the values need to assess convergence of a solve. It is typically instances are created in the solvers themselves, and initialised on construction of the solver. The method is\+\_\+converged is used to determine if a solve has converged. The criteria to satisfy convergence are as follows\+:
\begin{DoxyEnumerate}
\item i \texorpdfstring{$>$}{>} i\+\_\+min
\item i \texorpdfstring{$<$}{<} i\+\_\+max
\item $\vert$r$\vert$\+\_\+\{2, n\} \texorpdfstring{$<$}{<} epsilon
\item $\vert$r$\vert$\+\_\+inf \texorpdfstring{$<$}{<} 10.\+0$\ast$epsilon
\end{DoxyEnumerate}

where i is the number of iterations i\+\_\+min is the minimum number of iterations. i\+\_\+max is the maximum number of iterations. r is the residual vector of the linear system. epsilon is a user defined scalar tolerance. n ia the cardinality of the linear system (number of rows).
