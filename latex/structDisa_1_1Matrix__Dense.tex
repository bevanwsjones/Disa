\doxysubsection{Detailed Description}
\subsubsection*{template$<$std\+::size\+\_\+t \+\_\+row, std\+::size\+\_\+t \+\_\+col$>$\newline
struct Disa\+::\+Matrix\+\_\+\+Dense$<$ \+\_\+row, \+\_\+col $>$}

Mathematical matrix, of dimension \+\_\+row x \+\_\+col, where every matrix element has allocated has memory. 


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+row} & The number of rows in the matrix, if 0 a specialisation of the class is created. \\
\hline
{\em \+\_\+col} & The number of columns in the matrix, if 0 a specialisation of the class is created.\\
\hline
\end{DoxyTemplParams}
The \mbox{\hyperlink{structDisa_1_1Matrix__Dense}{Matrix\+\_\+\+Dense}} struct implements a mathematical matrix of nxm real numbers.

To avoid massive boiler plate the matrix inherits from std\+::array, which implies the matrix\textquotesingle{}s dimension is fixed at compile time. To obtain a dynamically allocated dense matrix the \+\_\+row and \+\_\+col value can be set to 0. In which case std\+::matrix is inherited, see below specialisation. Note that semi-\/static matrices are not supported, e.\+g. \+\_\+row = 1, \+\_\+col = 0.
