Specialization of the std\+::hash template for the \mbox{\hyperlink{classDisa_1_1Adjacency__Graph}{Disa\+::\+Adjacency\+\_\+\+Graph}} class. It calculates a hash value for the adjacency graph by XORing the hash values of its size, number of edges, and sizes of its first and last vertices. If the graph is empty, the hash value is set to zero.
