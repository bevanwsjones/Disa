\subsubsection*{template$<$std\+::size\+\_\+t \+\_\+size$>$\newline
struct Disa\+::\+Vector\+\_\+\+Dense$<$ \+\_\+size $>$}

Mathematical Vector, of dimension \+\_\+size, where every vector element has allocated has memory. 


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+size} & The size of the vector, if 0 a specialisation of the class is created.\\
\hline
\end{DoxyTemplParams}
The \mbox{\hyperlink{structDisa_1_1Vector__Dense}{Vector\+\_\+\+Dense}} struct implements a mathematical vector of real numbers. Disa does not make distinctions row and column vectors, as such this information is not stored.

To avoid massive boiler plate the vector inherits from std\+::array, which implies the vector\textquotesingle{}s dimension is fixed at compile time. To obtain a dynamically allocated dense vector the \+\_\+size value can be set to 0. In which case std\+::vector is inherited, see below specialisation.
