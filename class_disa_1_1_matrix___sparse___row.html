Detailed Description

template<typename _value_type, typename _index_type>
class Disa::Matrix_Sparse_Row< _value_type, _index_type >

Helper class, enables column iteration support.

Represents a row in a sparse matrix and provides access to its elements.

Template Parameters
_matrix_typeconst or non-const sparse matrix.

This class encapsulates the data and operations for a single row in a sparse matrix. It provides iterators for traversing the row's elements and methods for accessing specific elements.
