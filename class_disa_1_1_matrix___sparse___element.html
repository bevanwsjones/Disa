Detailed Description

template<typename _value_type, typename _index_type>
class Disa::Matrix_Sparse_Element< _value_type, _index_type >

Represents an element in a sparse (CSR) matrix.

Template Parameters
_value_typeThe type of the value in the matrix element.
_index_typeThe type of the row and column indices.

This class encapsulates a single element in a sparse matrix stored in Compressed Sparse Row (CSR) format. It provides read-write access to the element value and read-only access to the column index. The column value is read-only to maintain the validity of the CSR ordering.
