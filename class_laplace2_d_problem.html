Constructs a linear system to solve a 2D Laplace problem on a unit equi-spaced domain.

Parameters
[in]size_xThe number of nodes in one of the cardinal x-direction.

Consider an equi-spaced structured domain of size 1 + 2dx and the Laplace equation:

d^2phi/dx^2 + d^2phi/dy^2 = 1,

where dx = 1/(n_x - 1), and with zero as a dirichlet boundary condition. The discrete equation for a generic (non- boundary) node/vertex reads: A x = b | ... | | . | | dx^2 | | ... | | phi_{i - n_x} | | dx^2 | | ... | | . | | dx^2 | | ... | | phi_{i - 1} | | dx^2 | | ... -1 ... -1 4 -1 ... -1 ... | | phi_{i} | = | dx^2 |. | ... | | phi_{i + 1} | | dx^2 | | ... | | . | | dx^2 | | ... | | phi_{i + n_x} | | dx^2 | | ... | | . | | dx^2 |

Finally for this problem a zero initial condition is set.
