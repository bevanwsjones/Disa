Detailed Description

template<typename _entry_type, typename _index_type>
struct Disa::Matrix_Sparse_Element< _entry_type, _index_type >

Represents an element in a sparse matrix.

This struct stores the row index, column pointer, and entry pointer of an element in a sparse matrix. It provides member functions to access and modify the value, row index, column index, and entry of the element. It also overloads comparison operators for equality and inequality.

Template Parameters
_entry_typeThe type of the entry in the matrix element.
_index_typeThe type of the row and column indices.
