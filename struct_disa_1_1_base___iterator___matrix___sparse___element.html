Detailed Description

template<typename _entry_type, typename _index_type, bool _is_const>
class Disa::Base_Iterator_Matrix_Sparse_Element< _entry_type, _index_type, _is_const >

Base iterator for Matrix_Sparse_Element.

Template Parameters
_entry_typeThe type of the entry in the matrix element.
_index_typeThe type of the indices.
_is_constWhether this is a const iterator or not.

This class provides a bidirectional iterator for Matrix_Sparse_Element. It can be used to iterate over the elements of a sparse matrix in CSR format, typically row-wise although not exclusively.
