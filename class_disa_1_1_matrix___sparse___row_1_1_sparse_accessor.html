Detailed Description

template<typename _value_type, typename _index_type>
class Disa::Matrix_Sparse_Row< _value_type, _index_type >::SparseAccessor

Provides access and modification capabilities for sparse matrix elements.

This class acts as a proxy for accessing and modifying elements in a sparse matrix. It allows for implicit reading of existing elements and explicit insertion or modification of elements.
